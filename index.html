JZ_Offer_52两个链表的第一个公共节点

第一种常规做法，求出长度差值然后长链表先走断链表后走即可。

第二种比较有意思的代码，双指针不断寻找。
设交集链表长c,链表1除交集的长度为a，链表2除交集的长度为b，有

  • a + c + b = b + c + a
  • 若无交集，则a + b = b + a

常见排列问题

LC_31求解下一个排列

要分析清楚题意。
首先，从尾部向头部搜索，找到第一个比其后驱大的数（如果不存在，则说明没有下一个排列，可直接返回reverse后的数组）。然后，假设其位置为i-1，在(i,nums.size()-1]中寻找大于nums[i]的最小的数，并与其交换。最后对（i+1,end）进行排序即可。

LC_46全排列

标准的回溯题。不使用标记数组节省空间。

LC_47无重复的全排列

标准的回溯题。需要加判断以避免重复的排列出现。即if (i > 0 && nums[i] == nums[i - 1] && !vis[i - 1]) continue;。这句话规定了相同大小的数字在排列里的出场顺序，只有在前一个相同的数已经用过后，才能将当前数放进去。将!vis[i - 1]改为vis[i - 1]也有相同作用。
